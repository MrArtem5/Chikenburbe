# --О чем эта игра?--
Суть этой игры - "взломать" и получить все флаги из ядра чикенбурбе. Вы можете также получить секретные команды, который пригодятся для взлома. Главная цель - уничтожить ядро.

# --Как играть?--

1. **Скачай интерпретатор Lua** (версия 5.4 или 5.5). Официальный сайт: [lua.org](https://lua.org).
2. **Скачай репозиторий** (как архив или через `git clone`).
3. **Открой терминал** и перейди в папку со скриптом:
   - *Windows:* зажми `Shift` + правый клик по папке → «Открыть окно PowerShell», затем введи `lua main.lua`.
     Если путь содержит пробелы или русские буквы, используй кавычки: `cd "C:\Мои документы\Chikenburbe"`.
   - *Linux/macOS:* открой терминал, выполни `cd /путь/к/папке`, затем `lua main.lua`.
4. Внутри игры используй команды: `help`, `scan`, `disable_firewall`, `exploit`, `extract_code`, `extract_data`, `damage_kernel`, `show_flags`, `show_hidden` и другие.
5. Найди **все флаги**, уничтожь ядро (например, секретной командой `nuke`) и **победи**.
6. Скриншот победы можно выложить с хештегом `#чикенбурбе` в соцсеть `итд.com`.

дальше после запуска вы должны ознакомится с командами которые предоставлены (help, scan и тд.), а потом находить все эти флаги, а в после нахождения всех флагов вы должны уничтожить ядро (например с помощью секретной команды) и тогда вы победите.
